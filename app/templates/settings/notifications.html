✈️ Telegram Bot

Sends messages via a Telegram bot to a chat or channel. Create a bot with @BotFather, then get your Chat ID by messaging your bot and checking https://api.telegram.org/bot{TOKEN}/getUpdates.

Quick Setup Guide

  1. Message @BotFather on Telegram
  2. Send /newbot and follow the prompts
  3. Copy the API Token and paste it above
  4. Start a conversation with your bot or add it to a group
  5. Visit https://api.telegram.org/bot<TOKEN>/getUpdates
  6. Copy the chat.id value and paste it above
  7. Save and test!
