ℹ️ Import behaviour details
  • Existing alert rules are skipped unless "overwrite" is checked.
  • Settings are merged — only keys present in the file are updated.
  • REDACTED values in the file are never written to the database.
  • Alert history rows with duplicate IDs are skipped.
  • Imported users get a locked placeholder password and must reset via Admin → Users.
